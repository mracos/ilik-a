**Free Date and Age Calculator**

A clean, brutalist-inspired calculator for date math and age calculations. Add or subtract time, calculate exact ages, and plan dates with ease.

## Features

- **Date Calculator**: Add or subtract hours, days, weeks, months, or years from any date
- **Age Calculator**: Calculate precise age from any birth date
- **Quick Actions**: One-click buttons for common calculations (tomorrow, next week, etc.)
- **Flexible Input**: Type dates or use the date picker
- **Detailed Age Breakdown**: See age in years, months, weeks, days, hours, minutes, and seconds
- **Clean Interface**: Brutalist design with maximum usability
- **Zero Dependencies**: Pure HTML, CSS, and JavaScript - no frameworks needed

## Usage

### Date Calculator

1. **Set base date**: Choose your starting date (or click "Today")
2. **Enter offset**: Type a number to add or subtract (e.g., 29 or -10)
3. **Select unit**: Choose hours, days, weeks, months, or years
4. **Calculate**: Click Calculate to see the result date
5. **Quick buttons**: Use preset buttons for common date calculations

### Age Calculator

1. **Enter birth date**: Select your birth date using the date picker
2. **Calculate**: Click Calculate Age to see detailed age breakdown
3. **View results**: See age in multiple units (years, months, weeks, days, hours, minutes, seconds)

## Quick Start

Simply open `index.html` in any modern browser. No build process, no installation required.

```bash
git clone <repository-url>
cd ilik-a
open index.html
```
